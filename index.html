애드립 SDK 소개

애드립 SDK 는 특정 플랫폼에 의존적이지 않으며,
원하는대로 자신의 앱에 최적화하여 다양한 광고 플랫폼을 손쉽게 적용할 수 있도록 제작되었습니다.

애드립을 이용한 테스트 프로젝트를 다운로드 하시려면 위의 링크를 클릭해주세요.

또는 아래 링크에서 adlibr Library 만을 다운로드할 수 있습니다.

애드립에서 사용가능한 플랫폼은 아래와 같으며,
제휴 플랫폼을 이용하는 경우 별도의 리워드 포인트를 추가로 적립받을 수 있습니다.

제휴 플랫폼
 INMOBI (http://korea.inmobi.com)

제휴 플랫폼은 테스트 프로젝트안에 SDK를 포함한 모든 구현부가 미리 정의되어 있으며,
별도의 SDK 다운로드 및 적용이 불필요합니다.

일반 플랫폼
 * confirmed compatible with ad@m SDK 2.0.3
 * confirmed compatible with adcube SDK 1.6
 * confirmed compatible with admob SDK 6.0.0
 * confirmed compatible with cauly SDK 1.5.0
 * confirmed compatible with t-ad SDK 2.45.43.5
 * confirmed compatible with naverAdPost SDK 1.0
 * confirmed compatible with shallWeAd SDK 2.3

일반 플랫폼은 사용자의 편의에 의해 각 플랫폼 사이트에서 SDK 를 별도로 다운로드 받아 연동할 수 있으며, 정상적인 동작이 확인된 SDK 버전은 위와 같습니다.

실제로 각 플랫폼의 마이너 버전 업데이트가 일어나더라도 애드립 구현부의 변경없이 업데이트 된 SDK 만 새롭게 적용하면 그대로 사용가능하며, 최소한의 구현부 수정으로 큰 수정없이 적용할 수 있도록 구성되었습니다.

애드립을 발전시켜주세요

애드립은 열린 플랫폼으로 각 광고 플랫폼 구현부를 손쉽게 커스터마이징 할 수 있도록 제작되었습니다.
애드립 구현부는 github 를 통해 자유롭게 공유가 가능합니다. 애드립을 더 쓸모있는 플랫폼으로 만들기 위해 여러분의 참여를 기다립니다.
